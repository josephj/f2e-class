課程簡介

現今的網站開發需要考慮的更多，例如：瀏覽器兼容性 (Compatibility)、不換頁刷新 (JSONP / AJAX)、行動裝置瀏覽 (Mobility)、效能最佳化 (Performance)、資料的持續傳輸 (Comet/nodeJS) 以及整合 HTML5、CSS3 等新技術應用。已非昔日僅需視覺設計與程式設計師兩種角色可以處理，前端工程師 (Front End Engineer 簡稱 F2E) 的職位在像是 Facebook、Google、Yahoo!、百度等各大公司到新創 Start-up 的小公司都扮演了不可或缺的角色。

本課程由曾就職於 Yahoo!、前端開發經驗 6 年的 josephj 擔任講師。從 HTML/CSS/JavaScript 等基礎開始教起，讓學員對前端先能有開發正確的觀念。以製作「照片瀏覽、分享、下載」網站為範例整合 YUI JavaScript 函式庫與 nodeJS。讓學員在課程後都能擁有前端開發所具備的知識與技能。

學習目標

  • 上手 YUI 函式庫
  • 徹底瞭解 HTML/CSS/JavaScript
  • 能擁有前端開發所具備的基礎知識與技能

課程對象

  • 對前端開發有興趣、希望成為專業的前端開發工程師、瞭解 HTML/CSS、並有基本的程式開發底子。

課程內容

一、基本架構 - HTML (3 小時)

  • 課程介紹
  • 環境安裝 (jsfiddle.net, Notepad++, AppServ)
  • 語意與 HTML
  • 搜尋引擎最佳化
  • 照片列表規劃
  • 模組基礎

二、樣式之美 - CSS (3 小時)

  • 常用 CSS 屬性介紹與實作
  • CSS Hacks - 處理瀏覽器相容性
  • Reset CSS
  • CSS 排版
  • Compass - CSS 快速開發工具
  • 照片列表模組美化

三、賦予生命 - JavaScript (3 小時)

  • 常用 JavaScript 語法
  • 瞭解 JavaScript 物件導向基礎
  • 簡介 BOM - 操作瀏覽器
  • 簡介 DOM - 瞭解 JavaScript 事件、用程式操作及撰寫 HTML 與 CSS
  • 製作 JavaScript Bookmarklet
  • 利用 setInterval 實作 JavaScript 動畫

四、功力大增 - 使用 YUI 函式庫 (3 小時)

  • 瞭解 YUI、如何使用 YUI 官網
  • YUI 架構 - 用最快速的方式載入所需要的「模組」並且使用
  • YUI 工具快速上手
  • 使用 CSS Selector 取得 Node 與 NodeList
  • 有效率地綁定上千個、上萬個節點的事件
  • 五行做出炫麗製作動畫
  • 乾坤挪移 - 購物車拖拉
  • 照片網站的改進 - 將照片拖拉到下載區

五、厚實內力 - YUI 進階 (3 小時)

  • 不換頁載入 Facebook 的照片 IO, JSONP
  • 分離 HTML - 前端樣板引擎 Handlebars
  • 拖拉、批次上傳照片
  • Lightbox 的使用
  • 照片網站的改進 - 製作上傳功能與瀏覽照片

六、自我省視、行雲流水 - 除錯與效能 (3 小時)

  • Fiddler 的安裝與使用
  • 瀏覽器內建工具的使用
  • 提昇網站效能

七、飛鴿傳書 - nodeJS (3 小時)

  • Comet、Long Polling 等技術瞭解
  • 以 nodeJS 配合 Socket.IO 製作即時聊天室。
  • 照片網站的改進 - 有人上傳即有提示訊息。

八、放眼未來 - Mobility 與 HTML5 (3 小時)

  • Viewport Meta
  • HTML5 介紹
  • 讓照片網站可以在手機中被瀏覽
  • 結語 - 淺談前端工程師的價值、面試。
